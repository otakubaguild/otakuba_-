修正版テーマ素材アップロードパック

内容:
- presets/space/
  現在の presets.json が参照しているファイル名に合わせて、SF背景・敵画像を配置
- presets/magic/
  現在の presets.json が参照しているファイル名に合わせて、魔法学校背景・敵画像を配置
- presets.json
  space / magic の theme.assets を追加して、コンセプト切替時にスタート背景・討伐背景・BGM・マスター画像が反映されるよう修正

アップロード方法:
1. ZIPを解凍
2. GitHubリポジトリ直下にアップロード
3. presetsフォルダはそのまま追加/上書き
4. presets.json は上書き
5. 管理画面 → Game設定 → コンセプトで「宇宙・SF」「魔法学校」を選択

注意:
- 敵画像の一部は元画像の都合で背景込みです。動作確認は可能です。
- 本番で綺麗にしたい敵は、同じファイル名で透過PNGに差し替えればOKです。
